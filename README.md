Lagoona — Сеть отелей


Учебный проект по верстке многостраничного сайта для вымышленной сети отелей «Lagoona». Проект демонстрирует навыки работы с сетками, формами и декоративными элементами.


🔗 Живая ссылка на проект

👉 Посмотреть на GitHub Pages:

https://karinakit.github.io/Lagoona/


🌟 Особенности проекта

Pixel Perfect: 

Строгое соответствие дизайн-макету.

Интерактивные элементы:

Стилизованные состояния ссылок и кнопок (hover, focus, active).

Сложные формы бронирования номеров.

Кастомные чекбоксы и выпадающие списки.


Графика: 

Использование иконочных шрифтов и SVG-графики для четкости на любых экранах.


Сетки: 

Построение макета на основе Flexbox.


🛠 Технологии

HTML5 (Семантическая верстка);

CSS3 (Flexbox, позиционирование, кастомные свойства);

Normalize.css (для кроссбраузерности).
